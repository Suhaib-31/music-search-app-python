# Music Search App 🎵

## Overview

Music Search App is a Python application that allows users to search for songs and retrieve song information using the iTunes Search API.

The application displays song details including artist name, album name, release date, and preview URL.

---

## Features

- Search Songs
- Artist Information
- Album Information
- Release Date
- Song Preview URL
- API Integration

---

## Technologies Used

- Python 3
- Requests Library
- REST API
- JSON Parsing

---

## Installation

```bash
pip install requests
```
